# Shared Package

Tipos y utilidades compartidas entre frontend y backend.

## Descripción

Contiene:

- Tipos TypeScript comunes
- Constantes compartidas
- Utilidades reutilizables
- Esquemas de validación (si aplica)

## Estructura

- `src/index.ts`: Punto de entrada

## Uso

```typescript
import {} from /* exports */ "@mini-crm/shared";
```

## Supuestos

- TypeScript
- Se exportará como ESM
- Reutilizable en ambos apps
