# Web App

Frontend React para Mini CRM Tickets.

## Descripción

Interfaz de usuario para gestión de tickets personales.

## Estructura

- `src/`: Código TypeScript/React
- `public/`: Assets estáticos

## Desarrollo

```bash
# Instalar dependencias (pendiente)
npm install

# Ejecutar dev server (pendiente)
npm run dev
```

## Build

```bash
# Build para producción (pendiente)
npm run build
```

## Supuestos

- React 18+
- TypeScript
- Vite o Next.js (sin configurar aún)
- Tailwind CSS (pendiente)
